2 Mise en Route (exercice descriptives)

Les commandes exactes utilisees au debut de l’exercice de statistiques descriptives :

library(tidyverse)
library(ggthemes)            # themes graphiques supplementaires (theme_economist, etc.)
setwd("~/Documents/R")       # definir le repertoire de travail

# Importer un CSV : read.csv (R de base) ou read_csv (tidyverse) -- les deux marchent
# stringsAsFactors = TRUE convertit directement les colonnes texte en facteurs
reponses_socio <- read.csv("~/Documents/R/reponses_socio.csv",
                           encoding = "UTF-8", stringsAsFactors = TRUE)

View(reponses_socio)         # ouvre le tableau dans un onglet RStudio
Tipread.csv ou read_csv ?

read.csv() est la fonction de R de base ; read_csv() vient du tidyverse. Elles font la meme chose ici. read_csv() est un peu plus rapide et affiche un resume des colonnes ; read.csv() fonctionne sans charger de paquet. A l’examen, utilise celle vue en cours – les deux sont acceptees.

3 Statistiques Descriptives

3.1 Exploration rapide

str(df)        # types des colonnes
summary(df)    # min, Q1, mediane, moyenne, Q3, max par colonne

3.2 Tendance centrale et dispersion

mean(df$variable, na.rm = TRUE)      # moyenne
median(df$variable, na.rm = TRUE)    # mediane
sd(df$variable, na.rm = TRUE)        # ecart-type
var(df$variable, na.rm = TRUE)       # variance
IQR(df$variable, na.rm = TRUE)       # intervalle interquartile (Q3 - Q1)
range(df$variable, na.rm = TRUE)     # minimum et maximum
quantile(df$variable, na.rm = TRUE)  # min, Q1, mediane, Q3, max

3.3 Tableaux de frequences

# Comptage d'une variable categorielle
df %>% count(ma_variable)
df %>% count(ma_variable, sort = TRUE)

# Frequences en pourcentages
df %>%
  count(ma_variable) %>%
  mutate(pourcentage = n / sum(n) * 100)

# Tableau croise de deux variables categorielles
df %>% count(variable_1, variable_2)
table(df$variable_1, df$variable_2)

3.4 Tableau descriptif par groupe

df %>%
  group_by(variable_categorielle) %>%
  summarize(
    n          = n(),
    moyenne    = mean(variable_num, na.rm = TRUE),
    mediane    = median(variable_num, na.rm = TRUE),
    ecart_type = sd(variable_num, na.rm = TRUE)
  ) %>%
  ungroup()

3.5 Ordonner une variable categorielle

ordre <- c("Jamais", "Parfois", "Souvent", "Toujours")
df <- df %>% mutate(ma_variable = factor(ma_variable, levels = ordre))

3.6 Visualiser une relation (nuage de points + droite)

# Nuage de points avec droite de regression lineaire
ggplot(df, aes(x = variable_x, y = variable_y)) +
  geom_point() +
  geom_smooth(method = "lm")

3.7 Exporter un jeu de donnees

5 Test t

5.1 Test t a un echantillon (comparer a une reference)

# H0 : la moyenne vaut 100
t.test(df$variable, mu = 100)

5.2 Test t a deux echantillons (comparer deux groupes)

# Deux vecteurs separes
t.test(groupe_a, groupe_b)

# Une variable numerique selon une variable a 2 categories
t.test(variable_num ~ variable_groupe, data = df)

6 ANOVA (3 groupes ou plus)

# H0 : tous les groupes ont la meme moyenne
mon_anova <- aov(variable_num ~ variable_groupe, data = df)
summary(mon_anova)
# Test post-hoc : quelles paires de groupes different
TukeyHSD(mon_anova)

7 Correlation

# Coefficient seul (r entre -1 et +1)
cor(df$variable_1, df$variable_2, use = "complete.obs")

# Test de correlation : ajoute p-value et intervalle de confiance
cor.test(df$variable_1, df$variable_2)

8 Regression Lineaire

# Regression simple : Y explique par X
modele1 <- lm(variable_y ~ variable_x, data = df)
summary(modele1)
# Regression multiple : Y explique par plusieurs X
modele2 <- lm(variable_y ~ variable_x1 + variable_x2, data = df)
summary(modele2)
# Eviter la notation scientifique dans les sorties (ex. 2e-16)
options(scipen = 999)
# Graphiques de diagnostic du modele (4 plots en grille 2x2)
par(mfrow = c(2, 2))
plot(modele1)
# Regression par etapes : choisir le meilleur modele (minimise l'AIC)
full_model <- lm(variable_y ~ ., data = df)   # ~ . = toutes les autres variables
stepwise_model <- step(full_model, direction = "both")
summary(stepwise_model)

9 Aide-Memoire : Lire les Sorties R

9.1 Sortie de t.test()

Element Ce que c’est
t Statistique de test
df Degres de liberte
p-value < 0.05 -> difference significative
confidence interval Fourchette plausible de la difference
mean of x Moyenne(s) observee(s)

9.2 Sortie de summary(aov())

Element Ce que c’est
F value Statistique de l’ANOVA
Pr(>F) p-value globale ; < 0.05 -> au moins un groupe differe

9.3 Sortie de summary(lm())

Element Ce que c’est
Estimate Intercept et pente(s) ; signe = sens de l’effet
Std. Error Incertitude du coefficient
Pr(>|t|) p-value du coefficient ; < 0.05 -> predicteur significatif
Multiple R-squared Part de variance de Y expliquee par le modele
F-statistic p-value Le modele global est-il utile

9.4 Codes de significativite

Code Seuil
*** p < 0.001
** p < 0.01
* p < 0.05
. p < 0.1 (non significatif au seuil 5 %)

10 Choisir le Bon Test (rappel)

# 1 moyenne vs une reference
t.test(x, mu = valeur)

# 2 moyennes (2 groupes)
t.test(x ~ groupe, data = df)

# 3+ moyennes
aov(x ~ groupe, data = df)   # puis TukeyHSD()

# Lien entre 2 variables numeriques
cor.test(x, y)

# Modeliser / predire Y
lm(y ~ x1 + x2, data = df)
