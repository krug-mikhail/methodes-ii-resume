# Theorie : Enquete et Questionnaire

## 1. La Question de Recherche (RQ)

Une question de recherche scientifique doit etre :
- **Claire :** Eviter la confusion et le vague ; bien delimiter la recherche
- **Faisable :** Rester realiste par rapport aux ressources (personnelles, materielles, temps)
- **Pertinente :** Viser a mieux comprendre le phenomene plutot que de le decrire ; eviter les previsions et les jugements moraux

## 2. Operationnalisation

C'est le processus qui consiste a rendre un concept abstrait (ex: *Social Media Literacy*) mesurable a l'aide de variables concretes et de questions specifiques.

## 3. Niveaux de Mesure des Variables

Les variables peuvent etre classees selon 4 types essentiels :

| Type | Exemple | Proprietes |
|------|---------|------------|
| **Nominale** | Genre, Profession | Categories sans ordre |
| **Ordinale** | Frequence de publication ("jamais" a "tous les jours") | Categories avec ordre |
| **Continue** | Temperature en degres | Valeurs numeriques continues |
| **Discrete** | Nombre de followers | Valeurs numeriques entieres |

## 4. Types de Questions

Dans un questionnaire, on retrouve principalement :
- **Questions categorielles :** A choix multiple
- **Questions d'intensite :** Echelles mesurant une frequence ou un degre d'accord (ex: "Combien de fois...", Echelle de 1 a 5)
