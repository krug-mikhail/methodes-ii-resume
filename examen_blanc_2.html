1 · Statistiques descriptives
Code · Q1
Écris la commande qui calcule l'écart-type de la colonne trajet du data frame reponses_socio, en ignorant les valeurs manquantes.
Solution : sd(reponses_socio$trajet, na.rm = TRUE)
sd() = écart-type. na.rm = TRUE est indispensable dès qu'il peut y avoir des NA, sinon le résultat serait NA.
Code · Q2
Écris la commande (R de base) qui affiche un tableau croisé des variables genre et suisse du data frame reponses_socio.
Solution : table(reponses_socio$genre, reponses_socio$suisse)
table() avec deux variables produit un tableau a double entrée (effectifs croisés). En dplyr : count(reponses_socio, genre, suisse).
Code · Q3
Écris une chaîne dplyr qui regroupe reponses_socio par genre, puis calcule l'effectif n et la moyenne de note_uni (appelle cette colonne moy).
Solution : reponses_socio %>% group_by(genre) %>% summarize(n = n(), moy = mean(note_uni, na.rm = TRUE))
Schéma classique group_by + summarize. n() compte les observations de chaque groupe. summarise() (orthographe britannique) marche aussi.
Réponse courte · Q4
Pour la variable note_ecole, le 1er quartile vaut Q1 = 4.5 et le 3e quartile Q3 = 5.0. Quelle est la valeur de l'écart interquartile (IQR) ?
Solution : IQR = Q3 - Q1 = 5.0 - 4.5 = 0.5
L'IQR (intervalle interquartile) est la différence Q3 - Q1 : l'étendue des 50 % centraux des données. En R : IQR(reponses_socio$note_ecole, na.rm = TRUE).
Code · Q5
Crée un histogramme ggplot2 de la variable trajet de reponses_socio, avec une largeur d'intervalle de 10 et des barres remplies en "steelblue".
Solution : ggplot(reponses_socio, aes(x = trajet)) + geom_histogram(binwidth = 10, fill = "steelblue")
Pour une surface (barre), la couleur de remplissage est fill (et non color, qui ne colore que le contour). binwidth fixe la largeur des classes.
2 · Test t
Code · Q6
Écris un test t à un échantillon qui vérifie si la moyenne de l'échantillon scores_qi diffère de 100.
Solution : t.test(scores_qi, mu = 100)
Le test t à un échantillon compare la moyenne observée à une valeur de référence donnée par l'argument mu.
Interprétation · Q7
Une sortie de test t affiche t = 2.81, df = 49, p-value = 0.007. Au seuil de 5 %, rejette-t-on H0 ? (réponds par oui ou non)
Solution : Oui — on rejette H0.
p = 0.007 < 0.05 (et même < 0.01) → on rejette H0 : la différence est statistiquement significative.
Réponse courte · Q8
Pour un test t à un échantillon réalisé sur n = 30 observations, combien valent les degrés de liberté (df) ?
Solution : df = n - 1 = 30 - 1 = 29
Pour un test t à un échantillon, les degrés de liberté valent n - 1.
Code · Q9
Écris un test t à deux échantillons comparant les moyennes de tailles_f et tailles_h.
Solution : t.test(tailles_f, tailles_h)
R applique par défaut le Welch Two Sample t-test (ne suppose pas l'égalité des variances).
Interprétation · Q10
Un test t à deux échantillons donne un intervalle de confiance à 95 % de [-20.3 ; -2.4] pour la différence des moyennes. La différence est-elle significative ? (oui / non)
Solution : Oui — l'IC ne contient pas 0.
L'intervalle ne contient pas 0 : « aucune différence » est donc exclu → différence significative. (Si l'IC contenait 0, ce serait non significatif.)
Code · Q11
Écris le code (2 lignes) qui simule 50 valeurs suivant une loi normale de moyenne 105 et d'écart-type 15, de façon reproductible (utilise la graine 123).
Solution : set.seed(123) echantillon <- rnorm(50, mean = 105, sd = 15)
set.seed() fige le hasard (reproductibilité), rnorm(n, mean, sd) génère les valeurs.
3 · ANOVA
Code · Q12
Écris la commande qui réalise une ANOVA de body_mass_g selon species dans le data frame penguins, en stockant le résultat dans un objet nommé mon_anova.
Solution : mon_anova <- aov(body_mass_g ~ species, data = penguins)
Syntaxe : aov(variable_a_expliquer ~ variable_groupe, data = ...). On lit ensuite le résultat avec summary(mon_anova).
Code · Q13
L'ANOVA mon_anova est significative. Écris la commande du test post-hoc qui indique quelles paires de groupes diffèrent.
Solution : TukeyHSD(mon_anova)
TukeyHSD() compare chaque paire de groupes. On ne le lance qu'après une ANOVA significative.
Interprétation · Q14
summary() de l'ANOVA affiche F = 594.8, Pr(>F) < 2e-16. Au moins un groupe a-t-il une moyenne significativement différente ? (oui / non)
Solution : Oui — on rejette H0.
Pr(>F) < 2e-16 ≈ 0 < 0.05 → on rejette H0. Au moins un groupe diffère (l'ANOVA ne dit pas lequel — il faut Tukey).
QCM · Q15
Une ANOVA sur 4 groupes donne p < 0.001. Que peut-on affirmer avec certitude ?
  • Les 4 groupes ont des moyennes toutes différentes deux à deux
  • Au moins un groupe a une moyenne différente des autres — sans savoir encore lesquels
  • Aucun groupe ne diffère des autres
  • Le groupe 1 a la moyenne la plus élevée
Une ANOVA significative signifie qu'au moins un groupe diffère — pas que tous diffèrent, et pas lequel. Il faut un test post-hoc (Tukey) pour identifier les paires.
4 · Régression linéaire
Code · Q16
Ajuste une régression linéaire simple de mpg (variable à expliquer) par wt dans mtcars, en stockant le modèle dans modele1.
Solution : modele1 <- lm(mpg ~ wt, data = mtcars)
Syntaxe : lm(Y ~ X, data = ...). Y est la variable à expliquer, X la variable explicative.
Interprétation · Q17
Dans summary(modele1), le coefficient (Estimate) de wt vaut -5.34. Quand le poids wt augmente, la consommation mpg ____ ? (un mot)
Solution : elle diminue (de 5,34 en moyenne par unité de poids).
Le coefficient est la pente : signe négatif → quand X augmente, Y diminue. Ici chaque unité de wt en plus retire 5,34 à mpg.
Code · Q18
Ajuste une régression multiple : mpg expliqué par wt et hp, dans mtcars.
Solution : lm(mpg ~ wt + hp, data = mtcars)
On ajoute les variables explicatives avec +. Chaque coefficient s'interprète alors « toutes choses égales par ailleurs ».
Réponse courte · Q19
La sortie d'un modèle indique Multiple R-squared : 0.83. Quel pourcentage de la variance de la variable à expliquer le modèle capte-t-il ?
Solution : 83 %
Le R² se lit directement en pourcentage de variance expliquée : 0.83 → 83 %. Les 17 % restants viennent de facteurs hors du modèle.
Code · Q20
Écris le code (2 lignes) qui affiche les 4 graphiques de diagnostic du modèle modele1 dans une grille 2×2.
Solution : par(mfrow = c(2, 2)) plot(modele1)
par(mfrow = c(2, 2)) divise la fenêtre en grille 2×2 ; plot() sur un modèle lm affiche alors les 4 graphiques de diagnostic.
Code · Q21
Régression par étapes : écris le code (2 lignes) qui crée le modèle complet (mpg expliqué par toutes les variables de mtcars) puis applique step() dessus.
Solution : full_model <- lm(mpg ~ ., data = mtcars) stepwise_model <- step(full_model, direction = "both")
Le ~ . signifie « expliqué par toutes les autres variables ». step() retire/ajoute des variables pour minimiser l'AIC.
QCM · Q22
Modèle A : R² = 0.75 avec 1 variable. Modèle B : R² = 0.77 avec 8 variables. Lequel privilégier ?
  • Le modèle B, car son R² est plus élevé
  • Le modèle A : presque aussi explicatif avec 7 variables de moins (parcimonie)
  • Les deux sont strictement équivalents
  • On ne peut pas comparer deux modèles
Ajouter des variables fait toujours monter le Multiple R-squared. Le gain de B (+0.02) pour 7 variables de plus est dérisoire. On préfère le modèle parcimonieux (A) — c'est ce que confirment le R² ajusté et le critère AIC.
5 · Présentation des résultats
Réponse courte · Q23
Une étude trouve une corrélation r = .62 entre l'usage d'Instagram et l'anxiété. Cela prouve-t-il qu'Instagram cause l'anxiété ? (oui / non)
Solution : Non.
Corrélation n'est pas causalité. Un lien statistique, même fort, ne prouve jamais un lien de cause à effet (variable cachée, causalité inversée…).
Interprétation · Q24
La phrase suivante d'une section Résultats est incorrecte : « La corrélation r = .62 montre qu'Instagram cause de l'anxiété. » Réécris-la correctement (vérification indicative — compare aussi avec le modèle).
Modèle : « L'usage d'Instagram est associé à un niveau d'anxiété plus élevé (r = .62). » — on parle d'association, jamais de cause.
Il faut remplacer le verbe de causalité (« cause », « montre que ») par un terme d'association : « est associé à », « est lié à », « est corrélé avec ». Une corrélation décrit un lien, pas une cause.
