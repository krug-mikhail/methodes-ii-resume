1 · Statistiques descriptives
QCM · Q1
Quelle fonction affiche, pour chaque colonne d'un data frame, le minimum, Q1, la médiane, la moyenne, Q3 et le maximum ?
  • str()
  • summary()
  • head()
  • dim()
summary() donne le résumé statistique de chaque colonne. str() montre la structure (types), head() les 6 premières lignes, dim() le nombre de lignes/colonnes.
QCM · Q2
Pour la variable genre (Femme / Homme / Autre), quelle mesure de tendance centrale a du sens ?
  • La moyenne
  • La médiane
  • Le mode
  • L'écart-type
Le mode (catégorie la plus fréquente). genre est une variable nominale : on ne peut calculer ni moyenne ni médiane sur des catégories. L'écart-type n'est pas une mesure de centre.

Écris la commande qui calcule la moyenne de la colonne note_ecole du data frame reponses_socio, en ignorant les valeurs manquantes.
mean(reponses_socio$note_ecole, na.rm = TRUE)
# na.rm = TRUE indique d'ignorer les NA, sinon le resultat serait NA
QCM · Q4
Dans une entreprise, le salaire médian est 4'000 CHF et le salaire moyen 9'000 CHF. Que peut-on en conclure ?
  • Il y a forcément une erreur de calcul
  • La distribution est asymétrique : quelques très hauts salaires tirent la moyenne vers le haut
  • La moitié des employés gagne 9'000 CHF
  • L'écart-type est nul
La moyenne est sensible aux valeurs extrêmes. Quand moyenne ≫ médiane, la distribution est étirée vers les hautes valeurs. La médiane décrit alors mieux le cas typique.
QCM · Q5
Que calcule la fonction sd() ?
  • La somme
  • La variance
  • L'écart-type
  • La médiane
sd = standard deviation = écart-type. La variance se calcule avec var() (et variance = écart-type²).

2 · Test t (statistiques inférentielles I)
QCM · Q11
À quoi sert le test t ?
  • À compter des catégories
  • À comparer des moyennes
  • À créer un graphique
  • À trier un data frame
Le test t (de Student) évalue si une moyenne diffère d'une référence, ou si deux moyennes diffèrent significativement l'une de l'autre.
QCM · Q12
Que teste la commande t.test(scores_qi, mu = 100) ?
  • Si deux groupes ont la même moyenne
  • Si la moyenne de scores_qi diffère significativement de 100
  • Si scores_qi suit une loi normale
  • La corrélation entre scores_qi et 100
C'est un test t à un échantillon : on compare la moyenne de l'échantillon à une valeur de référence fixée par l'argument mu.
QCM · Q13
Une sortie de test t donne p-value = 0.1262. Au seuil de 5 %, on conclut :
  • On rejette H0 : la différence est significative
  • On ne rejette pas H0 : la différence n'est pas significative
  • Le test est invalide
  • H0 a 12,6 % de chances d'être vraie
0.1262 ≥ 0.05 → on ne rejette pas H0. Attention : la p-value n'est pas la probabilité que H0 soit vraie ; c'est la probabilité d'observer ces données si H0 était vraie.
QCM · Q14
Le même test, refait avec un échantillon plus grand, donne p-value = 0.007. On conclut :
  • On ne rejette pas H0
  • On rejette H0 : la différence est significative
  • L'échantillon est devenu trop petit
  • Le résultat est identique au précédent
0.007 < 0.05 (et même < 0.01) → on rejette H0. Le même écart devient significatif avec plus d'observations : c'est la puissance statistique. Un résultat non significatif sur petit échantillon ne prouve donc pas l'absence d'effet.
QCM · Q15
Que signifie l'hypothèse nulle (H0) ?
  • Il existe un effet ou une différence
  • Il n'y a aucun effet, aucune différence (tout est dû au hasard)
  • L'échantillon est vide
  • La p-value vaut zéro
H0 = « rien ne se passe ». C'est l'hypothèse qu'on cherche à rejeter. H1 (alternative) dit « il y a un effet ».
Code · Q16
Écris la commande qui réalise un test t comparant les moyennes de deux échantillons indépendants, tailles_f et tailles_h.
t.test(tailles_f, tailles_h)
# R applique par defaut le "Welch Two Sample t-test"
# Variante si les donnees sont dans un data frame :
# t.test(taille ~ genre, data = mon_df)
QCM · Q17
Laquelle de ces conditions n'est PAS requise pour appliquer un test t ?
  • Les données sont continues
  • Les observations sont indépendantes
  • La variable suit approximativement une loi normale
  • Les deux groupes ont exactement le même nombre d'observations
Le test t ne requiert pas des effectifs égaux. Conditions réelles : données continues, échantillonnage aléatoire, indépendance, normalité approximative, et homoscédasticité (variances similaires).
QCM · Q18
À quoi sert set.seed(123) avant un rnorm() ?
  • À fixer la taille de l'échantillon
  • À garantir que les nombres aléatoires soient les mêmes à chaque exécution
  • À définir la moyenne de la distribution
  • À importer un fichier de données
set.seed() fige le générateur aléatoire : rnorm() produira toujours la même séquence. Indispensable pour la reproductibilité des résultats.
QCM · Q19
Un test t contre mu = 100 donne un intervalle de confiance à 95 % de [98.6 ; 110.4]. Cela indique :
  • La différence avec 100 est significative
  • La différence avec 100 n'est pas significative — l'IC contient 100
  • La moyenne vaut exactement 100
  • Le test a échoué
Si l'intervalle de confiance contient la valeur de référence (100), on ne peut pas exclure « aucune différence » → résultat non significatif.
Code · Q20
Écris le code qui simule, avec rnorm(), 30 valeurs suivant une loi normale de moyenne 7.9 et d'écart-type 2 (résultat reproductible).

echantillon <- rnorm(30, mean = 7.9, sd = 2)
# rnorm(n, mean, sd) : n valeurs, loi normale
3 · ANOVA
QCM · Q21
Pour comparer les moyennes de 3 espèces de manchots, quel test utiliser ?
  • Un test t
  • Une ANOVA
  • Une corrélation
  • Un histogramme
Le test t ne compare que 2 moyennes. Pour 3 groupes ou plus, on utilise l'ANOVA (aov()).
Code · Q22
Écris la commande qui réalise une ANOVA de flipper_length_mm selon species dans le data frame penguins, et stocke le résultat dans anova1.
anova1 <- aov(flipper_length_mm ~ species, data = penguins)
summary(anova1)
# syntaxe : aov(variable_a_expliquer ~ variable_groupe, data = ...)
QCM · Q23
Le summary() d'une ANOVA affiche Pr(>F) <2e-16 ***. On conclut :
  • Aucune différence entre les groupes
  • Au moins un groupe a une moyenne significativement différente des autres
  • Tous les groupes ont exactement la même moyenne
  • Le test est invalide
Pr(>F) est la p-value globale. <2e-16 ≈ 0 < 0.05 → on rejette H0. L'ANOVA dit qu'au moins un groupe diffère, mais pas lequel.
QCM · Q24
Après une ANOVA significative, quel test indique quelles paires de groupes diffèrent ?
  • Un nouveau test t
  • Le test post-hoc de Tukey, TukeyHSD()
  • Une régression linéaire
  • La fonction summary()
TukeyHSD(anova1) compare chaque paire de groupes. On ne le lance qu'après une ANOVA significative.
QCM · Q25
Pourquoi préférer une ANOVA à plusieurs tests t deux à deux ?
  • L'ANOVA est plus rapide à taper
  • Enchaîner les tests t gonfle le risque d'erreur de type I
  • Le test t ne fonctionne pas dans R
  • L'ANOVA ne demande aucune condition
Chaque test t comporte 5 % de risque d'erreur de type I ; les multiplier accumule ce risque. L'ANOVA fait un seul test global et protège contre cette inflation.
4 · Régression linéaire
Code · Q26
Écris la commande qui ajuste une régression linéaire de mpg (variable à expliquer) par wt dans mtcars, et affiche le résumé du modèle.
modele1 <- lm(mpg ~ wt, data = mtcars)
summary(modele1)
# lm(Y ~ X, data = ...) : Y expliquee par X
QCM · Q27
Dans summary(lm(mpg ~ wt, data = mtcars)), le coefficient (Estimate) de wt vaut -5.34. Interprétation :
  • Chaque unité de poids en plus augmente mpg de 5,34
  • Chaque unité de poids en plus fait baisser mpg de 5,34 en moyenne
  • Le poids n'a aucun effet sur mpg
  • mpg vaut toujours -5,34
Le coefficient est la pente : signe négatif → quand wt augmente d'1 unité, mpg diminue de 5,34 en moyenne.
QCM · Q28
La sortie indique Multiple R-squared : 0.7528. Cela signifie :
  • La corrélation vaut 0,75
  • Le modèle explique environ 75 % de la variance de mpg
  • 75 % des données sont manquantes
  • La p-value vaut 0,75
Le R² mesure la part de la variance de la variable à expliquer captée par le modèle. Ici 75 % ; les 25 % restants viennent d'autres facteurs.
QCM · Q29
Que réalise lm(mpg ~ wt + hp, data = mtcars) ?
  • Une régression linéaire simple
  • Une régression linéaire multiple
  • Une ANOVA
  • Un test de corrélation
Il y a plusieurs variables explicatives (wt ET hp) → régression multiple. Chaque coefficient s'interprète « toutes choses égales par ailleurs ».
QCM · Q30
À côté du coefficient de wt, R affiche ***. Cela indique :
  • Le coefficient est erroné
  • p < 0.001 : le prédicteur est très hautement significatif
  • Il manque des données
  • Le coefficient vaut 3
Codes de significativité : *** p<0.001, ** p<0.01, * p<0.05, . p<0.1. Trois étoiles = très hautement significatif.
QCM · Q31
Que produit plot(modele1) après un lm() ?
  • Le résumé chiffré du modèle
  • 4 graphiques de diagnostic (résidus, Q-Q, etc.)
  • Un simple histogramme
  • La p-value du modèle
plot() sur un modèle lm affiche 4 graphiques de diagnostic : Residuals vs Fitted, Q-Q Residuals, Scale-Location, Residuals vs Leverage. Ils vérifient les conditions du modèle.
QCM · Q32
Dans la formule lm(y ~ x), comment s'appelle y ?
  • La variable explicative / le prédicteur
  • La variable à expliquer (dépendante)
  • Le résidu
  • L'intercept
y = variable à expliquer (dépendante). x = variable explicative (indépendante / prédicteur). Le ~ se lit « expliquée par ».
5 · Présentation des résultats
QCM · Q33
Une section Résultats affirme : « l'hypothèse est confirmée, p = 0.5 ». Quelle est l'erreur ?
  • Aucune, l'affirmation est correcte
  • p = 0.5 est très supérieur à 0.05 → le résultat est NON significatif, l'affirmation est contradictoire
  • Une p-value ne peut pas être positive
  • Il faudrait écrire « p = 0.5 % »
Une p-value de 0.5 (50 %) ne permet pas de rejeter H0. Annoncer « confirmé » est faux — c'est sans doute une faute de frappe pour p < .05.
QCM · Q34
« Une corrélation r = .62 montre qu'Instagram cause de l'anxiété. » Quelle est l'erreur ?
  • Aucune erreur
  • Corrélation n'est pas causalité : un lien statistique ne prouve pas un lien de cause à effet
  • r devrait toujours valoir 1
  • Il fallait faire un test t
Une corrélation, même forte, ne démontre jamais une causalité (variable cachée, causalité inversée…). Formulation correcte : « est associé à ».
QCM · Q35
Comment rapporter correctement une moyenne dans une section Résultats ?
  • La moyenne seule suffit toujours
  • Avec son écart-type (SD) et l'échelle utilisée
  • Avec la couleur du graphique
  • Uniquement sous forme de pourcentage
Exemple correct : « M = 3.76, SD = 0.68, sur une échelle de 1 à 5 ». Une moyenne sans SD ni échelle est ininterprétable.
QCM · Q36
Dans un rapport scientifique, que désignent les abréviations M et SD ?
  • Médiane et Somme des Données
  • Moyenne (Mean) et écart-type (Standard Deviation)
  • Maximum et Minimum
  • Mode et Statistique Descriptive
M = Mean = moyenne ; SD = Standard Deviation = écart-type. Notation standard (style APA).
